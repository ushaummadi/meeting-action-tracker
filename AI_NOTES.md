# AI Usage Notes

## Tools Used

- ChatGPT (for UI logic refinement, session state design, CRUD flow)
- Groq LLM (for action item extraction)

## What AI Helped With

- Prompt design for structured JSON extraction
- Improving Streamlit layout structure
- Designing SQLite schema
- Adding filtering and completion metrics

## What I Verified Manually

- All CRUD operations
- Data persistence
- Done status logic
- Completion percentage accuracy
- Error handling for LLM failures

## LLM Choice

Groq (Llama 3 model) was selected due to:
- Low latency
- Cost efficiency
- Fast inference speed
